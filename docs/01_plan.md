## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 소리볼륨 조절기 |
| 한줄 소개 | 마음대로 조절할 수 없는 볼륨 컨트롤러 — 친구에게 링크를 보내서 당황시키는 장난 웹앱 |
| Pain Point | 친구를 웃기고 싶은데 간단하게 링크 하나로 장난칠 수 있는 도구가 없음 |

## 타겟 사용자

- **주요 타겟**: "장난꾸러기 민수" (24세, 대학생) — 친구 모임 분위기 메이커
- **사용자 특성**: 카톡 단톡방에 이상한 링크를 공유하는 걸 좋아하고, 친구가 당황하는 반응을 보는 게 재미있는 타입
- **시장 규모**: 10~20대 MZ세대 모바일 유저 (설치 없이 링크로 바로 접근 가능한 웹앱)

**핵심 시나리오**: 민수가 친구한테 링크를 보냄 → 친구가 볼륨 조절 시도 → 마음대로 안 됨 → 민수가 반응 보며 웃음

## MVP 범위

> 2시간 내에 구현 가능한 핵심 기능만 선정하세요.

- [x] 핵심 기능 1: **랜덤 룰렛 볼륨** — 슬라이더를 조절하고 마우스를 놓는 순간 룰렛이 등장, 룰렛 결과값으로 실제 볼륨이 설정됨 (예상 20분, 난이도 중)
- [x] 핵심 기능 2: **반대로 볼륨** — 슬라이더를 올리면 볼륨이 내려가고, 내리면 올라감. UI는 정상 표시, 실제 소리는 반대 (예상 10분, 난이도 하)
- [x] 핵심 기능 3: **도망가는 슬라이더** — 마우스/터치가 가까워지면 슬라이더 thumb이 도망감, 잡으면 정상 동작 (예상 20분, 난이도 중)

**안 만들 것 (Out of scope)**:
- 로그인/회원가입
- 리액션 캡처/공유 기능
- 다양한 장난 모드 추가
- 결과 페이지

## 비즈니스 모델

- **수익 모델**: 무료 바이럴 서비스 → 광고 삽입 또는 프리미엄 장난 모드 유료화 가능
